Privacy Policy for OpenClaw Browser Relay Lite

Last updated: June 1, 2026

OpenClaw Browser Relay Lite is designed for local operation. This extension communicates only with localhost endpoints (127.0.0.1 / localhost) used by a locally running OpenClaw Gateway relay.

1. Data We Process

  • Configuration data: Relay port and gateway token entered by the user.
  • Runtime relay data: Debug session messages required to relay commands/events between Chrome and the local relay process.

2. How Data Is Stored

  • Configuration values are stored locally in Chrome extension storage on the user’s device.
  • The extension does not require a cloud account for core functionality.

3. How Data Is Shared

  • By default, the extension does not send data to external third-party servers.
  • Data flow is limited to local relay communication initiated by user action.

4. Advertising and Sale of Data

  • The extension does not sell personal data.
  • The extension does not use user data for advertising or profiling.

5. User Control

  • Users can remove stored settings by uninstalling the extension or clearing extension storage.
  • Users can disable or remove the extension at any time from Chrome extension settings.

6. Security

  • The extension uses token-based local authentication for relay access.
  • Access scope is restricted to localhost host permissions.
